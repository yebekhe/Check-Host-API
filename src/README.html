Ping Checker using Check-Host API

بررسی‌کننده پینگ با استفاده از API وبسایت Check-Host

This is a simple PHP script that checks the ping time to a specified host using the Check-Host API. The script fetches the ping data from 3 different nodes (Tehran, Shiraz, and Tabriz) and calculates their average values, converts them to milliseconds, and returns the results as a JSON object.

این یک اسکریپت PHP ساده است که زمان پینگ برای host مشخص شده را با استفاده از API وبسایت Check-Host بررسی می‌کند. این اسکریپت اطلاعات پینگ را از 3 گره مختلف (تهران، شیراز و تبریز) دریافت کرده، میانگین آنها را محاسبه، به میلی‌ثانیه تبدیل می‌کند و نتایج را به صورت یک شی JSON برمی‌گرداند.

Usage

نحوه استفاده

To use this script, simply call the PHP file with the host parameter set to the desired host. For example:

برای استفاده از این اسکریپت، با فراخوانی فایل PHP با پارامتر host، می‌توانید از آن استفاده کنید. به‌عنوان مثال:

https://vowrss-8080.csb.app/?host=google.com&accuracy=5
Parameter Description Required
host IP or website address Yes
accuracy Adjust accuracy of response No
پارامتر توضیحات الزامیست؟
host آدرس IP یا وب سایت بله
accuracy تنظیم دقت پاسخ خیر

If the ping is zero, the domain is filtered, to be sure, you can check the result by refreshing the script
